## 📖 Overview

This repository contains an end-to-end Exploratory Data Analysis (EDA)
of IPL match data. The project explores trends in team performance, toss
decisions, player achievements, and match outcomes using Python.

## ✨ Features

-   Data Cleaning & Preprocessing
-   Exploratory Data Analysis
-   Statistical Insights
-   Visualizations
-   Business Insights
